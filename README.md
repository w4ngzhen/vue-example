# vue-example

## 介绍

基于 Vue2 + elementUI 的项目，包含了日常开发中的众多基础代码模板、组件。

- 基础T型布局

- 经典Web页面入口布局

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
